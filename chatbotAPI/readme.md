## How to run the ChatBot?

</div>

1. Navigate to the ChatBot folder:
```
cd chatbotAPI
```

2. Train the dataset `intents.json`:
```
python training.py
```
or
```
python3 training.py
```

3. Run the ChatBot in console:
```
python chatbot.py
```
or
```
python3 chatbot.py
```
